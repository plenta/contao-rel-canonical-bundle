## System requirements

- PHP: `^7.4 || ^8.0`
- Contao: `^4.13` (and later)
   
   
### Install using Contao Manager

Search for **rel** or **canonical** and you will find this extension.

### Install using Composer

```bash
composer require plenta/contao-rel-canonical-bundle
```
